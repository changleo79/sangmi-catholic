# 상미성당 홈페이지

용인 상미성당 홈페이지 프로젝트입니다.

## 기술 스택

- React 18
- TypeScript
- Tailwind CSS
- Vite
- React Router

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 미리보기
npm run preview
```

## 프로젝트 구조

```
src/
  ├── components/    # 공통 컴포넌트
  ├── pages/        # 페이지 컴포넌트
  ├── App.tsx       # 메인 App 컴포넌트
  ├── main.tsx      # 엔트리 포인트
  └── index.css     # 전역 스타일
```
