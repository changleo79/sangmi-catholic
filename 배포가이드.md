# 상미성당 웹사이트 배포 가이드 (간단 버전)

## 🚀 가장 쉬운 방법: Vercel 사용 (5분 안에 완료!)

### 1단계: GitHub에 프로젝트 업로드

1. **GitHub 가입** (없으면): [github.com](https://github.com)에서 가입
2. **새 저장소 만들기**:
   - GitHub 로그인 후 우측 상단 `+` 버튼 → "New repository"
   - 저장소 이름: `sangmi-catholic` (또는 원하는 이름)
   - Public 선택
   - "Create repository" 클릭

3. **프로젝트 업로드** (터미널에서 실행):
```bash
# Git 초기화 (아직 안 했다면)
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/당신의계정/sangmi-catholic.git
git push -u origin main
```

### 2단계: Vercel로 배포

1. **Vercel 가입**: [vercel.com](https://vercel.com) 접속 → "Sign Up" → GitHub 계정으로 로그인

2. **프로젝트 배포**:
   - Vercel 대시보드에서 "Add New..." → "Project" 클릭
   - 방금 만든 GitHub 저장소 선택
   - 자동으로 설정이 감지됨 (그대로 두면 됨)
   - "Deploy" 버튼 클릭

3. **완료!** 🎉
   - 1-2분 후 배포 완료
   - `https://sangmi-catholic.vercel.app` 같은 주소가 생성됨
   - 이 주소를 누구에게나 공유하면 됩니다!

## 📝 다른 방법들

### 방법 2: Netlify 사용 (Vercel과 비슷)

1. [netlify.com](https://www.netlify.com) 가입
2. GitHub 저장소 연결
3. Build command: `npm run build`
4. Publish directory: `dist`
5. Deploy!

### 방법 3: 빌드 파일 직접 업로드

1. `npm run build` 실행 (이미 완료됨)
2. `dist` 폴더의 모든 파일을 웹 호스팅 서버에 업로드
3. 호스팅 업체에 React Router 설정 문의 (SPA 라우팅 필요)
