# 콘텐츠 업데이트 방법 (공지사항, 성당앨범)

이 프로젝트는 서버/관리자 페이지 없이 간단히 파일을 수정해 콘텐츠를 업데이트합니다.

## 1) 공지사항 올리기
파일: `src/data/notices.ts`

- 새 공지를 배열 맨 앞(최신)에 추가합니다.
- 형식:
```ts
{
  title: '제목',
  date: 'YYYY-MM-DD',
  summary: '간단 요약 (선택)',
  linkUrl: '/files/2025-11-bulletin.pdf' // 선택
}
```
- 저장하면 홈의 "최근 공지사항"과 `공지/소식` 페이지에 자동 반영됩니다.

## 2) 성당 앨범 올리기
폴더: `public/albums/연도-월-앨범명/`

- 사진 파일(JPG/PNG)을 위 경로에 폴더로 올립니다. 예) `public/albums/2025-11-성탄준비/001.jpg`
- 파일: `src/data/albums.ts` 에 앨범을 추가합니다.
- 형식:
```ts
{
  title: '앨범 제목',
  date: 'YYYY-MM-DD',
  cover: '/albums/2025-11-성탄준비/001.jpg',
  photos: [
    { src: '/albums/2025-11-성탄준비/001.jpg', alt: '설명1' },
    { src: '/albums/2025-11-성탄준비/002.jpg', alt: '설명2' },
  ]
}
```
- 저장하면 홈의 앨범 미리보기(일부)와 향후 앨범 페이지에서 표시됩니다.

## 참고
- PDF/파일 업로드는 `public/files/` 아래에 저장하고, `linkUrl: '/files/파일명.pdf'`로 연결하세요.
- 이미지/파일 이름은 영문/숫자/하이픈을 권장합니다.
- 변경 후 브라우저 새로고침(F5)으로 반영 확인하세요.
