# Vercel Pro vs 네이버 클라우드 비교 분석

## Vercel Pro 플랜 ($20/월) 제한사항

### 포함된 리소스
- **대역폭**: 월 1TB (빠른 데이터 전송)
- **서버리스 함수 호출**: 월 1,000만 회
- **빌드 시간**: 월 1,000시간
- **엣지 요청**: 월 1,000만 개
- **월 $20 상당의 사용 크레딧** (추가 리소스에 사용 가능)

### 스토리지 관련
- **Vercel Blob Storage** (별도 서비스):
  - 월 1GB 스토리지 포함
  - 월 10GB 데이터 전송 포함
  - 추가 스토리지: GB당 $0.023
  - 추가 데이터 전송: GB당 $0.05

### 제한사항
- **명시적인 스토리지 용량 제한 없음** (크레딧 기반)
- 대역폭 1TB 초과 시 추가 비용 발생
- 서버리스 함수 호출 1,000만 회 초과 시 추가 비용

## 현재 프로젝트 사용 현황

### 네이버 클라우드 Object Storage에 저장되는 데이터
1. **메타데이터 (JSON 파일)**
   - `metadata/albums.json` - 앨범 목록 및 메타데이터
   - `metadata/bulletins.json` - 주보 목록 및 메타데이터
   - `metadata/notices.json` - 공지사항
   - `metadata/recruitments.json` - 단체 모집
   - `metadata/faqs.json` - 자주 묻는 질문
   - `metadata/massSchedule.json` - 미사 시간표
   - `metadata/sacraments.json` - 성사 안내
   - `metadata/catechism.json` - 예비신자 교리학교
   - `metadata/backups.json` - 백업 데이터
   - **예상 용량**: 수 KB ~ 수십 KB (매우 작음)

2. **이미지 파일**
   - `albums/{albumId}/{filename}` - 앨범 원본 이미지
   - `albums/{albumId}/thumbnails/{filename}.webp` - 썸네일 (200x200, WebP 70%)
   - 주보 이미지 (JPG/PDF)
   - **예상 용량**: 
     - 원본 이미지: 평균 2-5MB/장 (고해상도)
     - 썸네일: 평균 10-30KB/장 (200x200 WebP)
     - 앨범 1개당 평균 10장 기준: 약 20-50MB (원본) + 100-300KB (썸네일)

3. **예상 총 용량 (현재 기준)**
   - 메타데이터: < 1MB
   - 앨범 이미지: 앨범 수 × 평균 용량
   - 주보 이미지: 주보 수 × 평균 용량
   - **예시**: 앨범 10개 (100장), 주보 50개
     - 메타데이터: ~100KB
     - 앨범 원본: ~200-500MB
     - 앨범 썸네일: ~1-3MB
     - 주보: ~50-250MB (PDF/JPG)
     - **총합: 약 300MB ~ 1GB**

## 비교 분석

### Vercel Pro만 사용하는 경우

#### 장점
1. **통합 관리**: 모든 것이 Vercel에서 관리됨
2. **글로벌 CDN**: 자동으로 전 세계에 배포
3. **간편한 설정**: 환경 변수만 설정하면 됨
4. **빠른 배포**: Git 연동으로 자동 배포

#### 단점
1. **스토리지 비용**: Vercel Blob Storage 사용 시
   - 기본 1GB 포함
   - 추가 스토리지: GB당 $0.023
   - 예: 10GB 사용 시 → $20 + (9GB × $0.023) = **약 $20.21/월**
2. **대역폭 제한**: 1TB 초과 시 추가 비용
   - 추가 대역폭: GB당 약 $0.05-0.10
3. **서버리스 함수 제한**: 1,000만 회 초과 시 추가 비용
4. **데이터 전송 비용**: 이미지 다운로드 시 대역폭 사용

### 네이버 클라우드 Object Storage 사용 (현재 방식)

#### 장점
1. **저렴한 스토리지 비용**
   - Object Storage: GB당 월 약 ₩10-20 (약 $0.01-0.02)
   - 예: 10GB 사용 시 → **약 $0.10-0.20/월**
2. **CDN 연동**: 네이버 클라우드 CDN 사용 가능
3. **국내 최적화**: 한국 사용자에게 빠른 속도
4. **무제한 확장**: 용량 제한 없음 (사용량 기반 과금)
5. **직접 관리**: 네이버 클라우드 콘솔에서 직접 파일 관리 가능

#### 단점
1. **별도 관리**: Vercel과 네이버 클라우드 두 곳 관리 필요
2. **환경 변수 설정**: 네이버 클라우드 인증키 등록 필요
3. **CDN 설정**: 별도로 CDN 도메인 설정 필요

## 권장사항

### 네이버 클라우드를 계속 사용하는 것이 좋은 경우
1. **이미 설정 완료**: 현재 네이버 클라우드가 잘 작동 중
2. **비용 효율**: 대용량 이미지 저장 시 더 저렴
3. **국내 최적화**: 한국 사용자에게 빠른 속도
4. **직접 관리**: 네이버 클라우드 콘솔에서 직접 파일 관리 가능
5. **확장성**: 용량 제한 없이 사용 가능

### Vercel Blob Storage로 전환을 고려할 수 있는 경우
1. **통합 관리**: 모든 것을 Vercel에서 관리하고 싶을 때
2. **소규모 프로젝트**: 1GB 이하의 작은 용량만 사용
3. **글로벌 서비스**: 전 세계 사용자를 대상으로 할 때
4. **간편함**: 별도 클라우드 서비스 관리가 부담스러울 때

## 결론

**현재 상황에서는 네이버 클라우드를 계속 사용하는 것을 권장합니다.**

### 이유:
1. **비용 효율**: 이미지가 많아질수록 네이버 클라우드가 더 저렴
2. **안정성**: 현재 잘 작동 중이며 문제없음
3. **국내 최적화**: 한국 사용자에게 빠른 속도
4. **확장성**: 용량 걱정 없이 사용 가능
5. **직접 관리**: 네이버 클라우드 콘솔에서 직접 파일 확인/관리 가능

### Vercel Pro 플랜의 장점:
- **대역폭 1TB**: 이미지 다운로드에 충분
- **서버리스 함수 1,000만 회**: API 호출에 충분
- **빌드 시간 1,000시간**: 배포에 충분

**결론**: Vercel Pro는 호스팅과 서버리스 함수에 사용하고, 스토리지는 네이버 클라우드를 계속 사용하는 **하이브리드 방식**이 가장 효율적입니다.
