# 상미성당 홈페이지 기획 요약

참고: 명동대성당 사이트 구조를 간소화해 소형 본당에 맞게 적용 ([명동대성당 홈페이지](http://www.mdsd.or.kr/)).

## 정보 구조(IA)
- 홈(Home)
  - 히어로(성당 이미지, 환영 카피)
  - 빠른 링크(미사와 성사, 공지, 성당소개)
  - 최근 공지 3건
  - 미사 시간 요약
  - 앨범 미리보기 4장
- 성당소개(About)
  - 본당 소개, 사목 방침(간단 요약)
  - 연락처/주소/오시는 길 링크
- 공지/소식(News)
  - 공지사항 목록(간단 카드형)
  - 단체 모집(필요 시)
  - 본당 주보(PDF 링크/추후 업로드 위치)
- 미사와 성사(Mass)
  - 미사 시간 상세
  - 성사 안내(세례, 고해, 견진, 혼인, 병자)
  - 예비신자 교리학교 안내
- 성당업무(Office)
  - 사무실 운영시간, 연락처
  - 자주 묻는 질문(간단 3~5개)
- 오시는 길(Directions)
  - 약도/지도 링크(구글 지도)
  - 주차/대중교통 안내(간단 텍스트)

## 디자인 톤
- 색상: 블루 계열과 중성 그레이(부담스럽지 않은 톤)
- 배경: 사진과 얇은 오버레이로 따뜻한 분위기
- 카드형 레이아웃으로 간결하게

## 초기 콘텐츠(예시)
- 공지 3건(행사 안내, 미사 시간 변경, 교리학교 개강)
- 미사 시간(평일/주일 기본형)
- 사무실 시간(화~금, 토/주일, 점심시간) — 실제 운영에 맞게 수정 예정
- 지도: 구글 지도 링크 자리

## 향후 확장
- 앨범(슬라이더/갤러리 상세)
- 주보 업로드(파일 관리)
- Q&A 폼(이메일 연동)
